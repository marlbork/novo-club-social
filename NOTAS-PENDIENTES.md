# Pendientes para subir a IONOS

- Revisar el formulario de contacto antes de publicar.
- Sustituir el `mailto:info@novoclubsocial.com` por un formulario real de IONOS si está disponible.
- Confirmar que los mensajes llegan correctamente a `info@novoclubsocial.com`.
- Mantener el email visible en la web como vía directa de contacto.
- Antes de publicar, decidir si las nuevas imágenes de Unsplash se dejan externas o se descargan/localizan en `assets` para que la web no dependa de enlaces externos.
